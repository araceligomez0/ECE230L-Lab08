## Lab Questions

### In plain English describe the function and use of a multiplexer.
A multiplexer is a device that chooses a signal from various inputs and sends it to a single output. The way that this happens is by using the Enable and Select characteristics of the multiplexer. Their main use is to create a network that allows multiple signals to share only one line of communication. In the end, this helps that amount of hardware that is used in a digital system.

### In plain English describe the function and use of a demultiplexer.
A demultiplexer is a type of deive that takes in only one input signal and sends it to many outputs. Demultiplexers are used to help send date from one source to more than one desitnation. 

### What other uses might these circuits have? (Think Shannon’s)
The other uses that these circuits have is a way to implement Boolean logic functions without the use of individual logic gates. The multiplexers and demultiplexers are able to help simplify the logic functions with the Select attribute. In the end, a circuit with many logic gates can be simplified into a much smaller and cheaper design with the use of multiplexers and demultiplexers.
